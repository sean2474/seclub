# SE클럽(seclub.kr) 웹사이트 리메이크 프로젝트 PRD

## 1. 프로젝트 개요

### 목적
SE클럽(태안둘레길캠핑장) 웹사이트를 현대적인 디자인과 기능으로 리메이크하여 사용자 경험을 개선하고 예약 시스템을 효율화합니다.

### 배경
현재 웹사이트는 숫자 기반 URL 구조, 이미지 최적화 부족, 고객 지원 기능 미흡 등 다양한 개선이 필요합니다. 프라이빗 힐링 공간이라는 브랜드 아이덴티티는 유지하면서 기술적, 디자인적 개선이 필요합니다.

### 목표 사용자
- 가족 단위 여행객
- 연인/친구 단위 여행객
- 단체 여행객(워크숍, 회사 연수 등)
- 반려동물과 함께하는 여행객

## 2. 기술 스택 및 아키텍처

### 프론트엔드
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- React 기반 컴포넌트 구조

### 백엔드
- Next.js API Routes
- Supabase (데이터베이스, 인증, 스토리지)

### 호스팅
- Vercel (CI/CD 자동화)

### 데이터베이스
- Supabase (PostgreSQL)

## 3. 핵심 기능 요구사항

### 3.1 메인 페이지
- 고해상도 히어로 이미지/슬라이더
- 간략한 소개 및 핵심 가치 제안
- 주요 CTA 버튼("예약하기")
- 시즌별 프로모션 배너

### 3.2 숙박시설 소개
- 풀빌라, 자쿠지 단독주택, 호텔형 객실 상세 정보
- 시설별 고품질 이미지 갤러리
- 요금 정보(주중/주말, 성수기/비수기)
- 예약 가능 여부 표시

### 3.3 캠핑장 소개
- 캠핑 구역 위치도
- 구역별 특징 및 편의시설 정보
- 이용 요금 안내
- 예약 절차 설명

### 3.4 부대시설 소개
- 수영장, 스파/월풀, 바비큐장 등 정보
- 관광농원, 조각공원, 전망대 소개
- 사진 갤러리

### 3.5 고객 후기/리뷰 시스템
- 사용자 리뷰 작성 및 조회 기능
- 별점 시스템
- 사진 첨부 기능
- 관리자 답변 기능

### 3.6 예약 시스템
- 실시간 예약 가능 확인
- 날짜 및 객실/캠핑사이트 선택
- 예약자 정보 입력 및 결제
- 예약 확인 메일/문자 발송
- 예약 취소/변경 기능

### 3.7 관리자 페이지
- 예약 관리(조회/수정/취소)
- 콘텐츠 관리(이미지, 가격, 설명 등)
- 리뷰 관리(승인/거부/답변)
- 통계 대시보드(예약률, 매출 등)

### 3.8 고객 지원
- FAQ 페이지
- 문의 폼(이름, 이메일, 메시지)
- 실시간 채팅 지원(선택사항)

## 4. 비기능적 요구사항

### 4.1 성능
- 페이지 로딩 시간 1.5초 이내
- 이미지 최적화(WebP 포맷, 압축)
- Lazy Loading 적용

### 4.2 보안
- HTTPS 적용
- 사용자 데이터 암호화
- 관리자 인증 및 권한 관리

### 4.3 접근성
- WCAG 2.1 AA 준수
- 모바일 최적화(반응형 디자인)
- 다양한 기기 호환성

### 4.4 SEO
- 메타 태그 최적화
- 구조화된 데이터(schema.org) 적용
- 사이트맵 생성

## 5. 디자인 요구사항

### 5.1 스타일 가이드
- 브랜드 컬러: 해변과 자연을 연상시키는 컬러 팔레트
- 타이포그래피: 모던하고 가독성 높은 폰트
- 여백과 공간 활용으로 고급스러운 느낌 제공

### 5.2 UI/UX 디자인
- 직관적인 내비게이션 구조
- 모바일 우선 디자인
- 명확한 CTA 버튼
- 일관된 디자인 시스템 적용
